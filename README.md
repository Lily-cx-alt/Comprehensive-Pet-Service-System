> # 综合性宠物服务平台

## 1. 项目概述

### 项目目的
本项目旨在打造一个一体化的综合性宠物服务平台，成为宠物主人的贴心伴侣。系统致力于解决当前宠物服务领域中存在的领养流程不透明、服务预约分散、用品购买不便以及宠物情感陪伴缺失等痛点。通过整合宠物领养、服务预约、用品商城、AI智能交互及健康档案管理等核心功能，为宠物主人提供便捷、高效、一站式的宠物全生命周期服务体验。

### 目标用户群体
本系统的核心目标用户主要分为三类：

- **普通宠物主人**：可以浏览宠物信息、申请领养、预约寄养及医疗美容等服务、购买宠物用品、与宠物AI聊天以及管理宠物的健康档案。
- **商家用户**：负责管理服务项目、处理用户预约订单以及管理商品库存。
- **系统管理员**：负责用户权限管理、领养申请审核、宠物信息管理、首页轮播图配置、公告发布以及流浪动物求助信息的处理。

## 2. 功能介绍

### 模块一：用户管理
- **用户注册**：支持新用户通过手机号或邮箱注册账号。
- **用户登录**：支持账号密码登录，采用JWT实现无状态身份认证。
- **信息维护**：用户可以修改个人资料（头像、昵称、联系方式等），管理宠物档案（添加、编辑、删除名下宠物信息）。

### 模块二：宠物领养管理
- **领养浏览**：用户可以浏览待领养的宠物列表，查看宠物的详细资料（品种、年龄、健康状况等）。
- **领养申请**：用户可在线提交领养申请，填写申请信息。
- **申请审核**：管理员对领养申请进行审核，审核通过后宠物状态变更为“已领养”，形成完整的领养流程闭环。

### 模块三：宠物服务预约
- **服务浏览**：支持寄养、医疗、美容、训练等服务的分类展示与详情查看。
- **在线预约**：用户可选择服务类型、时间段进行在线预约。
- **订单管理**：管理员可管理服务类型、为预约分配资源，并跟踪服务状态（待处理、进行中、已完成、已取消）。

### 模块四：宠物用品商城
- **商品浏览**：支持宠物用品的分类展示、搜索和详情查看。
- **下单购买**：用户可将商品加入购物车并生成订单，完成购买流程。
- **库存与支付管理**：商家可管理商品库存；系统集成余额支付功能，通过数据库事务保证扣款与订单更新的原子性。

### 模块五：宠物AI聊天
- **拟人化对话**：用户可选择自己名下的宠物，与大模型（讯飞星火大模型API）进行拟人化智能对话。
- **个性化回复**：AI的回复会结合宠物的真实信息（如名字、品种、性格）进行个性化生成，增强互动趣味性和情感陪伴体验。

### 模块六：用户与权限管理
- **角色区分**：系统支持普通用户、商家和管理员三种角色。
- **动态权限**：根据登录角色动态加载不同的功能菜单与操作权限。
- **系统管理**：管理员可进行用户权限管理、公告发布、首页轮播图配置以及流浪动物求助信息的处理。
